# 摘要

这篇为知笔记介绍 PyTorch 神经网络的基本骨架 `nn.Module`。原文说明 nn 是神经网络的缩写，截图示例演示了继承 `nn.Module` 后将 input 加一再输出，是一个最简单的模块使用案例。

# 核心观点

1. `nn` 是 neural network 的缩写。
2. PyTorch 中自定义网络通常继承 `nn.Module`。
3. 原文示例只是把 input 加一后输出，用来说明基本骨架。

# 我的理解

这篇笔记的重点是建立 PyTorch 网络模块的入口概念：网络不是散装函数，而是继承 `nn.Module` 的对象。具体结构和层会在后续卷积层、池化层、线性层和 Sequential 中逐步扩展。

# 可复用方法

- 自定义网络时先继承 `nn.Module`。
- 在模块内部写清楚输入到输出的计算过程。
- 用极简例子先确认模块调用流程，再增加真实网络层。
