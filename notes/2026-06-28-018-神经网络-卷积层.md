# 摘要

这篇为知笔记介绍 PyTorch 神经网络中的卷积层。原文指出，如果输入通道为 1、输出通道为 2，则会有两个卷积核；还记录了创建神经网络代码、查看输出形状，以及将输入和输出图像显示到 `SummaryWriter` 中。原文还提到输出为六个通道时，为了显示成三个通道，需要用 `reshape` 处理。

# 核心观点

1. 输出通道数会影响卷积核数量。
2. 创建卷积网络后，需要查看输出形状。
3. `SummaryWriter` 可用于显示输入和输出图像。
4. 当输出通道数不适合显示时，可用 `reshape` 调整。

# 我的理解

这篇笔记把前一篇的卷积操作升级到神经网络层。卷积层不仅要关注计算，还要关注输入输出通道数和输出张量形状，否则后续可视化或接入下一层都会出错。

# 可复用方法

- 设置卷积层时先明确输入通道和输出通道。
- 跑一次前向传播，打印输出形状。
- 用 TensorBoard 查看卷积前后图像，必要时调整 reshape。
