# 摘要

这篇为知笔记介绍目标检测的基本分类和目标位置表示。原文正文写到“目标检测算法分类”，截图将算法分为“两步走的目标检测”和“端到端的目标检测”：前者先做区域推荐再做目标分类，代表包括 R-CNN、SPPNet、Fast R-CNN、Faster R-CNN；后者采用一个网络一步到位，代表包括 YOLO 和 SSD。另一张截图围绕目标检测的任务、位置表示和 bounding box 展开。

# 核心观点

1. 两步走目标检测先进行区域推荐，再进行目标分类。
2. 端到端目标检测使用一个网络完成检测流程。
3. 原文指出，当需要检测多个物体时，输出坐标不是简单的左上角和右下角两个坐标。
4. 截图中把 ground truth bounding box、predicted bounding box、预测分类和位置作为目标位置表示的核心概念。

# 我的理解

这篇笔记是在为后续 R-CNN 系列和 YOLO/SSD 做概念铺垫。基于原文和截图可推断，课程先强调目标检测比图像分类多了“在哪里”的问题，因此需要同时处理类别和位置，且多目标场景会让输出结构更复杂。

# 可复用方法

- 学习目标检测时先区分两阶段方法和端到端方法。
- 先理解 bounding box 表达，再看具体模型结构。
- 对比 ground truth box 和 predicted box，理解训练和评估时的位置误差来源。
- 将 R-CNN 系列和 YOLO/SSD 分别放到两类算法框架中记忆。
