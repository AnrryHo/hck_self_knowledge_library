# 摘要

这份材料总结矩阵相乘的特点：矩阵相乘通常不能交换顺序，符合结合律；单位矩阵与任何矩阵相乘仍得到原矩阵。

# 核心观点

1. 矩阵相乘不能交换顺序矩阵相乘符合结合律单位矩阵，任何矩阵与单位矩阵相乘都等于任何矩阵，与单位矩阵相乘是符合交换律的，但是两个单位矩阵的纬度不一定相等

# 图片识别补充

第一页上半部分图示强调矩阵乘法一般不满足交换律：通常 `A x B` 不等于 `B x A`，并且有些情况下一个方向能乘，另一个方向维度不匹配不能乘。页面中给出小矩阵例子说明顺序改变会改变结果。

第一页下半部分图示说明矩阵乘法满足结合律：`A x B x C` 可以先算 `A x B`，也可以先算 `B x C`，只要维度合法，最终结果相同。页面底部还引入单位矩阵，说明单位矩阵类似数字 1，矩阵与相容维度的单位矩阵相乘仍得到原矩阵。第二页继续展示单位矩阵例子和 `A x I = I x A = A` 的直觉。
