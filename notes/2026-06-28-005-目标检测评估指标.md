# 摘要

这篇为知笔记记录目标检测的评价指标。原文提到 mAP 平均准确率、准确率与召回率。截图中可以辨认出 IoU 交并比、平均精确率 mean average precision、mAP，以及每个类别 AP 计算后再求平均的说明。

# 核心观点

1. IoU 交并比用于衡量预测框与目标框位置重合程度，截图中标注其取值在 0 到 1 之间。
2. mAP 是目标检测评价中的核心指标之一。
3. 截图中显示 mAP 与各类别 AP 的平均有关。
4. 原文同时提醒需要理解准确率与召回率。

# 我的理解

目标检测评估不仅要判断分类是否正确，还要判断位置是否准确。基于截图可推断，IoU 主要服务于位置判断，AP/mAP 则把不同类别的检测效果综合起来评价。

# 可复用方法

- 先用 IoU 判断预测框与真实框是否匹配。
- 再结合准确率和召回率理解 AP。
- 多类别目标检测中，用各类别 AP 的平均值理解 mAP。
- 复习模型效果时，把分类正确性和定位准确性分开检查。

这篇笔记适合为目标检测项目建立评估口径，可与 R-CNN 系列算法笔记一起复习，判断算法改进是否体现在 mAP 或检测质量上。
