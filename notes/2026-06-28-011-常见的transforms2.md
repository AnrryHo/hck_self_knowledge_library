# 摘要

这篇为知笔记记录常见 transforms 的第二部分，覆盖 `Resize`、`Compose`、`RandomCrop` 和 TensorBoard 显示。原文说明 `Resize` 传入宽高时可重新定义图片大小，传入单个数值时按最长边缩放；`Compose` 可组合多个转换；`RandomCrop` 支持指定宽高或单个正方形尺寸，并可多次裁剪后显示。

# 核心观点

1. `Resize` 可以按宽高或单个尺寸缩放图片。
2. `Resize` 后图片类型不变，需要再用 `ToTensor` 转成 tensor。
3. `Compose` 用于把多个 transforms 按顺序组合。
4. `RandomCrop` 用于随机裁剪图片。
5. 使用 TensorBoard 可以观察图像变换结果。

# 我的理解

这篇笔记强调 transforms 的组合顺序。基于原文可确认，`Compose` 的输入必须与第一个 transform 匹配，后续每一步也要接收上一步输出的类型。

# 可复用方法

- 缩放后如果要输入模型，继续使用 `ToTensor` 转换类型。
- 使用 `Compose` 时按输入输出类型安排顺序。
- 用 `RandomCrop` 做数据增强前，确认裁剪尺寸是否适合原图。
- 将多次随机裁剪结果写入 TensorBoard 检查效果。
