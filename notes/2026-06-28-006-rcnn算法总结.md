# 摘要

这篇为知笔记总结 R-CNN 的效果和缺点。原文写到“精度较高，速度很慢，步骤繁琐”。截图可辨认出 R-CNN 在 VOC2007 数据集上的平均精度 mAP 达到 66%，同时列出缺点：训练阶段多、训练耗时、占用磁盘空间大、处理速度慢，以及候选区域经过 crop/warp 固定大小会影响图片形状。

# 核心观点

1. R-CNN 的优点是精度较高，截图中提到 VOC2007 上 mAP 达到 66%。
2. R-CNN 缺点之一是训练阶段多，包括微调网络、训练 SVM 和训练边框回归器。
3. R-CNN 会产生较大的特征文件，占用磁盘空间。
4. 处理速度慢，截图中提到使用 GPU 和 VGG16 处理单张图像仍需要较长时间。
5. crop/warp 固定候选区域大小可能导致图片形状变化。

# 我的理解

这篇笔记强调 R-CNN 的价值和问题并存：它提升了目标检测精度，但分阶段流程和重复计算带来很高成本。后续 SPPNet 和 Fast R-CNN 的改进方向，正是减少重复卷积、减少磁盘中间文件和整合训练流程。

# 可复用方法

- 总结 R-CNN 时同时记住“精度较高”和“速度很慢”两个判断。
- 从训练流程、磁盘空间、推理速度、候选区域变形四个角度分析 R-CNN 的缺点。
- 学习后续模型时，对照这些缺点理解改进目标。
