# 摘要

这篇为知笔记介绍现有网络模型的使用和修改。原文记录可以使用 `add_module` 方法添加模型层，示例是在现有 `vgg16_true` 模型中添加一层线性层；还记录了将现有模型的层进行修改。

# 核心观点

1. 现有模型可以继续添加新层。
2. `add_module` 可用于给模型添加模块。
3. 原文示例围绕 VGG16 模型添加线性层。
4. 除了添加层，也可以修改模型已有层。

# 我的理解

这篇笔记指向迁移学习或模型复用的基础操作：不一定从零搭网络，可以基于已有模型增加或替换最后几层来适配新任务。

# 可复用方法

- 先加载现有模型。
- 使用 `add_module` 添加需要的新层。
- 根据任务输出维度修改已有层。
- 修改后检查模型结构是否符合预期。

# 项目关联

这篇笔记适合连接模型保存读取、完整训练套路和 GPU 训练，用于已有模型的复用与微调。
