# 摘要

这篇为知笔记介绍 DataLoader 的使用。原文区分 dataset 和 dataloader：dataset 表示数据集及数据位置，dataloader 将数据加载到容器如神经网络中。示例中 `batch_size=64` 表示一次取出 64 张图片，组成一个 data，其中 64 个 image 组成 `imgs`，64 个 target 组成 `targets`。原文还提到可用 `writer.add_images` 将图片显示到 TensorBoard。

# 核心观点

1. Dataset 负责描述数据及其位置。
2. DataLoader 负责按批次加载数据。
3. `batch_size=64` 表示一次加载 64 个样本。
4. 一个 batch 中通常包含图片和目标标签。
5. `writer.add_images` 可用于批量显示图片。

# 我的理解

DataLoader 是从“数据集定义”进入“训练循环”的接口。它把单个样本组织成 batch，让模型训练可以按批次迭代。

# 可复用方法

- 先准备 dataset，再用 DataLoader 包装。
- 根据显存和训练需求设置 `batch_size`。
- 在调试阶段用 `add_images` 查看一个 batch 的图像是否正确。

这篇笔记是 PyTorch 训练套路中的数据加载环节，可连接 torchvision 数据集、模型训练循环和 TensorBoard 可视化。
