# 摘要

这篇为知笔记介绍 torchvision 中数据集的使用，重点是 datasets 与 transforms 的联合使用。原文示例使用 CIFAR10 数据集，并说明下载数据集后，可以通过 `transforms=dataset_transform` 将数据统一转换成 tensor。

# 核心观点

1. `torchvision.datasets` 可用于下载和加载数据集。
2. 原文示例使用 CIFAR10 数据集。
3. 数据集可以和 transforms 联合使用。
4. `transforms=dataset_transform` 用于指定数据转换流程。

# 我的理解

这篇笔记把 transforms 从单张图片处理扩展到整个数据集加载流程。通过在 dataset 创建时传入 transform，可以让每个样本在读取时自动完成预处理。

# 可复用方法

- 使用 torchvision 数据集时先选择目标数据集。
- 定义 `dataset_transform`，例如转换为 tensor。
- 创建 dataset 时通过 `transforms` 参数绑定转换流程。
- 后续再交给 DataLoader 批量加载。

# 项目关联

这篇笔记连接 transforms 和 DataLoader，是 PyTorch 图像训练数据管道的中间环节。

# 需要复习的知识点

- CIFAR10 数据集加载流程。
