# 摘要

这篇为知笔记记录 TensorBoard 中 `add_image` 的使用。原文指出 `add_image` 可以传入多种数据类型，截图示例中使用的是 numpy array 类型，并且需要设置 `dataformats`，相关说明可查看注释。

# 核心观点

1. `add_image` 用于把图像写入 TensorBoard。
2. 输入图像可以是 numpy array 等多种数据类型。
3. 使用 numpy array 时，需要关注 `dataformats` 设置。

# 我的理解

这篇笔记是在补充 TensorBoard 图像可视化的输入格式问题。可确认的信息是 `add_image` 并不只接受 tensor，还能接受 numpy array；但具体代码参数需要回查 raw 中截图。

# 可复用方法

- 使用 `add_image` 前先确认输入图像的数据类型。
- 若输入是 numpy array，检查 `dataformats` 是否与数组维度一致。
- 遇到显示异常时，优先回看 `add_image` 的参数注释。
