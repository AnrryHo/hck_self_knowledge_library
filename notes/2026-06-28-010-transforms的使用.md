# 摘要

这篇为知笔记介绍 PyTorch/torchvision 中 transforms 的基本使用。原文说明 transforms 用于对图片进行变换，使用方式是“创建工具，使用工具”；还提到可以利用 OpenCV 将图片读取为 numpy array，深度学习计算需要 tensor 数据类型，并可通过 `tensorboard --logdir=logs` 查看图像。

# 核心观点

1. transforms 是图像变换工具，需要先创建再应用。
2. OpenCV 读取图像后可以得到 numpy array 数据。
3. 深度学习计算需要 tensor 数据类型。
4. TensorBoard 可以用 `tensorboard --logdir=logs` 启动并查看图像。

# 我的理解

这篇笔记是在说明图像从原始文件进入模型前需要经过数据类型和格式转换。transforms 的位置在数据预处理环节，既能服务模型输入，也能配合 TensorBoard 做可视化检查。

# 可复用方法

- 读取图片后先确认当前数据类型。
- 使用 transforms 把图像转换为模型需要的 tensor。
- 需要查看中间图像时，将结果写入 TensorBoard 后用 `tensorboard --logdir=logs` 打开。

# 项目关联

这篇笔记是 PyTorch 图像数据处理链路的入口，可连接常见 transforms、torchvision 数据集和 DataLoader 使用笔记。
