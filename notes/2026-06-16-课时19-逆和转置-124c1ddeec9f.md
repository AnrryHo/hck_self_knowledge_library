# 摘要

这份材料介绍逆矩阵和转置矩阵：只有方阵才可能有逆矩阵，矩阵与其逆相乘得到单位矩阵；转置可理解为沿对角线翻转。

# 核心观点

1. 逆矩阵:只有方形矩阵才具有逆矩阵，当矩阵A与另外一个矩阵A'相乘，得到一个单位矩阵，那么这个矩阵就是矩阵A的逆矩阵在octave中，可以使用函数求逆矩阵一个方形矩阵如果没有其逆矩阵，那么称为奇异矩阵或者退化矩阵转置矩阵:可以想象成一个矩阵以一个45度对角线进行翻转，得到的新矩阵就是转置矩阵

# 图片识别补充

第一页上半部分图示用数字倒数类比逆矩阵：普通数字有 `3 * 3^-1 = 1`，矩阵中对应关系是 `A * A^-1 = I`。图中强调只有方阵才可能有逆矩阵，且并非所有方阵都有逆；没有逆的矩阵称为奇异矩阵。

第一页下半部分图示解释转置矩阵：把矩阵沿左上到右下的主对角线翻转，原来的行变成列，原来的列变成行。若 `A` 是 `m x n`，则 `A^T` 是 `n x m`，元素关系是 `B_ij = A_ji`。
