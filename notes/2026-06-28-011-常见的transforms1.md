# 摘要

这篇为知笔记记录常见 transforms 的第一部分。原文强调要关注图片的输入和输出，说明 `__call__` 是类的内置函数，可通过实例对象调用；还记录了 `ToTensor` 的使用，以及 `Normalize` 归一化/标准化，会把输入标准化到 -1 到 1 之间。

# 核心观点

1. 使用 transforms 时要关注每一步的输入和输出类型。
2. `__call__` 让实例对象可以像函数一样被调用。
3. `ToTensor` 用于将图片转换成 tensor。
4. `Normalize` 用于归一化或标准化输入。

# 我的理解

transforms 组合中最容易出错的是前后类型不匹配。这篇笔记提醒每个变换都有预期输入和输出，后续 Compose 组合时必须保持顺序合理。

# 可复用方法

- 每加一个 transform，都确认其输入类型和输出类型。
- 先用 `ToTensor` 完成张量化，再考虑归一化。
- 把 `__call__` 理解成 transforms 实例被调用时真正执行的入口。
