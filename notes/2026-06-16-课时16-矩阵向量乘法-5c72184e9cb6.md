# 图片识别补充

第一页图示说明矩阵向量乘法的维度规则：`A` 是 `m x n` 矩阵，`x` 是 `n x 1` 向量，结果 `y` 是 `m x 1` 向量。计算时用矩阵每一行与向量做对应元素相乘再求和，得到结果向量中的一个元素。

第二页图示把矩阵向量乘法用于房价预测：矩阵的每一行代表一套房子的特征，参数向量代表假设函数中的系数，矩阵乘以参数向量后一次性得到多套房子的预测价格。这说明矩阵向量乘法可以把多个样本的预测向量化。
