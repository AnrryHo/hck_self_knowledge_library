# 摘要

这篇为知笔记介绍神经网络中的最大池化。原文记录 `MaxPool2d` 参数：核心数可以设置，步长 `stride` 的默认值是核心数大小；`ceil_mode` 是否为 true 会决定最大池化时是否保留周边没有值的情况。原文还包含最大池化代码和输出截图。

# 核心观点

1. `MaxPool2d` 的核心大小可以设置。
2. `stride` 默认值与核心大小一致。
3. `ceil_mode` 会影响边界区域是否被保留。
4. 最大池化可以通过代码和输出结果验证。

# 我的理解

最大池化用于在保留主要响应的同时降低空间尺寸。原文更关注 PyTorch 参数含义，尤其是 `stride` 和 `ceil_mode` 对输出尺寸和边界处理的影响。

# 可复用方法

- 设置 `MaxPool2d` 时先明确 kernel size。
- 不显式设置 `stride` 时，记住它默认等于 kernel size。
- 边界结果不符合预期时检查 `ceil_mode`。
- 用小输入跑一次输出，验证池化行为。
