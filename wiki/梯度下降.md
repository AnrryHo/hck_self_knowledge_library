# 梯度下降

# 摘要

本主题页整理吴恩达机器学习课时 10、11、12、28、29 中关于梯度下降的内容。当前结论来自已入库笔记和 PDF 文本抽取，公式图示未完整抽取时需回查原 PDF。

# 核心概念

- 梯度下降通过循环更新 theta 参数，使代价函数逐步降低。
- theta0、theta1 等参数需要同步更新，不能在同一轮中混用新旧参数。
- 导数或偏导数提供更新方向；当斜率趋近于 0 时，参数变化会变小。
- 学习率 alpha 类似下降速率，过小会慢，过大可能跨过最低点。
- 线性回归场景下，课程材料指出代价函数没有局部最优问题，最优解只有一个。
- 多元梯度下降把单变量更新推广到多个特征；特征缩放可以改善收敛速度。
